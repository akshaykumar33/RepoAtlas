# Security Policy

RepoAtlas takes security seriously. We value the input of security researchers and the open-source community to help keep our software safe.

## Supported Versions

Only the latest active release branch of RepoAtlas receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

**Do NOT report security vulnerabilities through public GitHub issues.**

Instead, please report security issues privately by emailing the maintainers or submitting a Security Advisory on GitHub.

When reporting a vulnerability, please include:

1. Description of the vulnerability and its potential impact.
2. Steps to reproduce or proof-of-concept code.
3. Affected package(s) and version(s).

Maintainers will acknowledge receipt within 48 hours and work with you to resolve and release a patch promptly.
