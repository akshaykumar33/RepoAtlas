Initial v0.1.0 release of RepoAtlas developer platform.

Includes:

- High-performance recursive repository scanning engine with `.gitignore` parsing & project framework detection
- Multi-theme renderer supporting ASCII, Unicode, VSCode Explorer, Material Theme, and Nerd Font styles
- 13 Export plugins (TXT, Markdown, HTML, JSON, YAML, XML, CSV, DOCX, PDF, SVG, PNG, Mermaid, PlantUML)
- Commander.js CLI with live `--watch` preview mode and `doctor` system diagnostics
- VSCode extension with Explorer context menu support and Webview preview
- Official GitHub Action for automated `PROJECT_STRUCTURE.md` and `MERMAID.md` workflow updates
