# @repoatlasdev/cli

> Command Line Interface (CLI) for RepoAtlas.

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

## Installation

```bash
npm install -g @repoatlasdev/cli
```

## Quick Start

```bash
repo-atlas generate
repo-atlas generate --theme vscode --sort size
repo-atlas preview
repo-atlas doctor
```
