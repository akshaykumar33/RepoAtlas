# Changelog

## v0.1.0 - Initial Release

- Core scanning engine with `.gitignore` and max depth support
- Project detector for Next.js, React, Vue, Express, NestJS, and monorepos
- Pluggable renderers: ASCII, Unicode, VSCode style, Material Theme, Nerd Font
- 13 Exporters: TXT, MD, HTML, JSON, YAML, XML, CSV, DOCX, PDF, SVG, PNG, Mermaid, PlantUML
- Commander CLI suite with `--watch`, `--compact`, `--color`, `--sort`, `--only`
- Docusaurus documentation website & 4 interactive tools
