# Frequently Asked Questions (FAQ)

### How is RepoAtlas different from standard `tree`?

Standard `tree` only prints directory strings. RepoAtlas is a complete developer platform: it parses `.gitignore`, detects frameworks (Next.js, React, Node), formats in 5 themes (VSCode, Material, Nerd Font), exports to 13 formats (HTML, PDF, SVG, Mermaid, DOCX), and runs as a CLI, VSCode extension, or GitHub Action.

### Does RepoAtlas send any telemetry or code data?

No. RepoAtlas is 100% open-source and operates strictly locally on your machine.
