# Icon Packs

RepoAtlas provides 7 icon pack presets:

- `emoji`: Standard colored emojis (`📁`, `📂`, `📄`, `📘`, `📦`)
- `unicode`: Classic Unicode symbols (`🗀`, `🗁`, `🗋`)
- `plain` / `ascii`: Plain text brackets (`[D]`, `[F]`)
- `vscode`: VSCode Developer Icons
- `material`: Material Icon Theme Set
- `nerd`: Nerd Font Developer Glyphs
