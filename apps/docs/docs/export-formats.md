# Export Formats

RepoAtlas supports 13 export output formats via `@repo-atlas/exporters`:

- `txt`: Plain text tree
- `md`: Markdown code block document
- `html`: Standalone interactive dark-mode HTML document
- `json`: Serialized JSON tree data
- `yaml`: Structured YAML hierarchy
- `xml`: Standard XML tree representation
- `csv`: Flattened CSV table (`path`, `name`, `type`, `sizeBytes`, `modifiedAt`)
- `docx`: Formatted OpenXML Word processing document
- `pdf`: Native PDF stream document
- `svg`: Scalable Vector Graphics SVG tree diagram
- `png`: PNG image data stream
- `mermaid`: Mermaid `graph TD` diagram syntax
- `plantuml`: PlantUML `@startuml` component diagram
