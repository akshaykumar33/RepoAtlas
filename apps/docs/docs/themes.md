# Themes

RepoAtlas supports 5 themes out of the box.

## Theme Presets

### 1. Unicode (`unicode`)

Box drawing tree connectors with emoji icon resolution.

```
рҹ“Ӯ root
в”ңв”Җв”Җ рҹ“Ғ src
в”Ӯ   в””в”Җв”Җ рҹ“ҳ index.ts
в””в”Җв”Җ рҹ“Ұ package.json
```

### 2. ASCII (`ascii`)

Standard ASCII character connectors.

```
[D] root
|-- [D] src
|   `-- [F] index.ts
`-- [F] package.json
```

### 3. VSCode Style (`vscode`)

VSCode Explorer style connectors with VSCode icons.

```
рҹ“Ӯ root
в”ңв”Җ рҹ“Ғ src
в”Ӯ  в””в”Җ оҳЁ index.ts
в””в”Җ оңҳ package.json
```

### 4. Material Theme (`material`)

Material Theme connectors with vibrant color palettes.

### 5. Nerd Font (`nerd-font`)

Developer glyph icon mappings (`\uf115`, `\uf15c`, `\ue718`).
